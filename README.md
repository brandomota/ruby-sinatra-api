### RUBY SINATRA API

---

REST API built in Ruby, using Sinatra Framework.

---

### ENVIRONMENT
 - Ruby 3.x+

---

### HOW TO RUN

- install dependencies: `bundle install`
- start rackup server: `rackup -p [PORT]`
